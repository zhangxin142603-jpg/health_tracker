# v1.0.1 - 发布准备与配置优化

**发布日期**: 2026-03-06
**版本类型**: 补丁更新 (Patch Release)
**兼容性**: 完全向后兼容

## 🎯 用户体验改进
- **应用显示名称**: 从 "health_tracker" 更改为中文 "身心记录"，提升本地化体验

## 🔒 安全性增强
- **生产环境签名**: 配置正式 Android 签名密钥 (`my_release_key`)
  - 密钥算法: 2048位 RSA
  - 签名算法: SHA256withRSA
  - 有效期: 25年 (2026-2051)
  - 证书信息: CN=张新, OU=个人开发, O=个人, L=广州, ST=广东, C=CN
- **移除测试密钥**: 删除原有的 debug 签名配置，确保发布安全

## 🛡️ 开发流程优化
- **Git 保护**: 将 `key.properties` 和 `release-key.jks` 添加到 `.gitignore`，防止密钥文件意外提交
- **配置标准化**: 统一签名配置管理，便于团队协作

## 📦 技术变更
- **Gradle 配置**: 更新 `android/app/build.gradle.kts` 支持生产环境签名
- **版本管理**: 应用版本号从 `1.0.0+1` 更新至 `1.0.1+2`
